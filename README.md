# Student GitHub Starter Kit

A beginner-friendly starter pack for university students learning coding, Git, and GitHub.  
This kit gives you a clear structure for your assignments, projects, notes, and weekly study workflow.

---

## What's Inside

### 1. Essential Guides
- Git cheat sheet (simple and student-friendly)
- How to use this kit step by step
- VS Code setup walkthrough
- Recommended folder structure for uni projects

### 2. Templates
Inside the templates folder:
- Assignment notes template (Markdown)
- Project report template (Markdown)
- Weekly study log template

### 3. Starter Code
Inside starter-code:
- HTML/CSS/JS starter project
- Python starter file

### 4. Bonus Tools
Inside extras:
- VS Code setup guide
- Uni project folder structure guide

---

## Who This Is For
Students who:
- Are new to Git, GitHub, or coding
- Want a clean structure for assignments
- Prefer simple, practical study tools
- Want to start uni with confidence

---

## How to Use This Kit

1. Download the repo or clone it  
2. Open it in VS Code  
3. Start your weekly study log  
4. Use the templates for each assignment  
5. Add projects inside the starter-code folder  
6. Commit your work regularly using the cheat sheet
