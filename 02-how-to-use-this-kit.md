# How to Use This Kit

This kit gives you a clean structure for studying, coding assignments, and GitHub workflows.

---

## 1. Download or clone the kit

### Option A — Download ZIP
1. Go to the GitHub repo
2. Click the green Code button
3. Choose Download ZIP
4. Extract the folder

### Option B — Clone (recommended long-term)
git clone <REPO_URL>

---

## 2. Open it in VS Code

1. Open VS Code
2. File → Open Folder
3. Select the starter-kit folder
4. You’ll see:
   - templates
   - starter-code
   - extras
   - this guide

---

## 3. Start your weekly study workflow

1. Go to templates/
2. Copy weekly-study-log-template.md
3. Rename it to something like:
   week-01-study-log.md
4. Fill it at the start and end of each week

This keeps your semester organised.

---

## 4. Use the assignment and project templates

### For assignments
1. Copy assignment-notes-template.md
2. Rename it to the actual assignment name
3. Track your requirements, research, plan, and final checks

### For projects
1. Copy project-report-template.md
2. Rename it to match your project
3. Fill it in as you build your work, not at the end

---

## 5. Use the starter code

### Web (HTML/CSS/JS)
1. Go to starter-code/html-css-js-starter
2. Open index.html using Live Server
3. Edit and experiment

### Python
1. Go to starter-code/python-starter/main.py
2. Run it:
   python main.py

---

## 6. Create your own version for each subject

For every new unit:
1. Copy the whole kit folder
2. Rename it e.g. COMP1010-starter
3. Use:
   - weekly logs
   - assignment templates
   - project report templates

This builds strong habits across the semester.

---

## 7. Back up everything to GitHub

Use the Git commands from the cheat sheet:

- git add .
- git commit -m "Your update"
- git push

Your work will always stay safe and backed up.

---

This kit becomes more useful the more you reuse it each semester.
