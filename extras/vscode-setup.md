# VS Code Setup Guide for Beginners

This quick guide helps new students set up Visual Studio Code for coding assignments and university projects.

---

## 1. Install VS Code
Download it here:  
https://code.visualstudio.com/

---

## 2. Recommended Extensions
Install these from the Extensions tab:

### For all students
- Prettier – Code Formatter
- GitHub Pull Requests & Issues
- GitLens
- Markdown All in One

### For Web Dev
- Live Server
- HTML CSS Support

### For Python
- Python (Microsoft)
- Pylance

---

## 3. Helpful Settings
Search for "Format on Save" in Settings and turn it on.  
This keeps your files clean automatically.

---

## 4. Using the Terminal
Open the built-in VS Code terminal:
- Windows: CTRL + `
- Mac: CMD + `

You can:
- run Python files  
- run Git commands  
- check folder structures

---

## 5. Common Actions
### Open a folder
File → Open Folder → select your project.

### Create files
Right-click → New File.

### Run Python scripts




### Use Live Server (HTML/CSS/JS)
Right-click index.html → "Open with Live Server".

---

You're now ready to build projects with this kit.
