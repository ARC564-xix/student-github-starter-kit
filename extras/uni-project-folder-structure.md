# Recommended Uni Project Folder Structure

Use this simple structure for any coding assignment or project.

project-name/

├─ docs/ # diagrams, notes, report drafts

├─ src/ # your code files

├─ tests/ # test files (if required)

├─ assets/ # images or media

└─ README.md # project overview


---

## Why this works

### 1. Clean
Markers and group members can find things fast.

### 2. Reusable
Use the same structure for all units.

### 3. Professional
Matches how real developers organise projects.

---

## How to Use This Template
1. Create a new folder for your project  
2. Copy this structure  
3. Add your report notes in docs  
4. Add your code in src  
5. Commit regularly using the Git cheat sheet  
6. Push to GitHub to back up your work

---

This setup keeps your work organised and easier to maintain.
